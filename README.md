## Làm xong push lên với step sau

kiểm tra tình trạng git với :

```
git status
```

1. Thêm vào các thay đổi vào chuẩn bị commit

```
git add .
```

2. Thêm lời nhắn vào thay đổi

```
git commit -m "feat: I have add 1 more feature"
```

3. Đẩy code lên remote

```
git push origin
```
